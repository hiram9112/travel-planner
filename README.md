# Travel Planner

Personal full-stack project, built to practice modern web development with a realistic frontend and backend stack.

The goal of this project is to simulate a real-world application, following professional practices and technologies commonly required for junior web developer roles.

---

## Tech Stack

### Frontend
- Vue 3 (Composition API)
- TypeScript
- Vite
- HTML5 / CSS3

### Backend
- Node.js
- TypeScript
- Express
- REST API

### Database
- PostgreSQL
- Prisma ORM
